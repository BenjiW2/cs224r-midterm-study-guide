Answer Key

Use this after attempting the exam. The explanations are intentionally short, matching what you should be able to produce under time pressure.
1. MDPs, Returns, And Reward Shaping

Correct: A, B, and D. Arbitrary dense rewards and changing γ can change optimal policies.

2. Imitation Learning

True, False, True, True, True, True. The key traps are compounding errors for behavior cloning and action averaging for multimodal continuous actions.

3. Policy Gradients

Correct: A, B, D, and E. A baseline must not depend on the sampled action if it is being subtracted inside the score-function term.

4. Actor-Critic, GAE, And PPO

Correct: A, B, C, and E. PPO clipping limits the surrogate objective incentive, not the literal parameter movement.

5. Q-Learning And DQN

All five statements are true.

6. Offline RL, IQL, And RLHF

Correct: A, B, C, D, and F. E is false: the reward model is usually a training-time scoring model, while the updated language model is deployed.

7. Discounted Returns

G0 = 0 + 0.5·2 + 0.52(-1) + 0.53·4 = 0 + 1 - 0.25 + 0.5 = 1.25.

G1 = 2 + 0.5(-1) + 0.52·4 = 2 - 0.5 + 1 = 2.5.

8. REINFORCE Update Sign

If action A: g = d/dp log p = 1/p = 2.5. Update: p_new = 0.40 + 0.01·5·2.5 = 0.525.

If action B: g = d/dp log(1-p) = -1/(1-p) = -1/0.6 = -1.667. Update: p_new = 0.40 + 0.01·5·(-1.667) ≈ 0.317. Good B rollout lowers probability of A, making B more likely.

9. Actor-Critic Advantage

Â = 3 + 0.9·8 - 7 = 3.2. Positive advantage, so increase probability of the sampled action.

10. PPO Clipped Objective

Sample 1: rA=1.3·4=5.2, clipped ratio is 1.2, clipped term 4.8, surrogate uses 4.8.

Sample 2: rA=0.7·4=2.8, clipped term 0.8·4=3.2, surrogate uses 2.8. No improvement cap is active because the ratio moved in a bad direction for positive advantage.

Sample 3: rA=0.7·(-4)=-2.8, clipped term 0.8·(-4)=-3.2, surrogate uses -3.2. For negative advantage, decreasing probability too much is capped.

11. Q-Learning And Double DQN Targets

Ordinary DQN: target-network max is 7, so y=1+0.9·7=7.3.

Double DQN: online network selects left because 8 is largest. Target network evaluates left as 4. So y=1+0.9·4=4.6.

Loss using Double DQN target: (9-4.6)2=19.36.

12. n-Step Target

y = 1 + 0.5·0 + 0.52·2 + 0.53·6 = 1 + 0 + 0.5 + 0.75 = 2.25.

13. IQL Expectile And Policy Extraction

Average: (2+5+6)/3 = 13/3 ≈ 4.33.

Advantages with V̂(s)=5.5: A is 2-5.5=-3.5, B is 5-5.5=-0.5, C is 6-5.5=0.5.

Weighted behavior cloning copies C most strongly. This is safer offline because it only weights dataset actions instead of maximizing over arbitrary unseen actions whose Q-values may be wrong.

14. RLHF Preference Model

Score difference: 3-1=2. sigmoid(2) is greater than 0.5, so the reward model assigns greater-than-even probability to the preferred response.

A large KL penalty discourages the updated policy from drifting too far away from the frozen reference model.
