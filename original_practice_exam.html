Answer Key

Use this after attempting the exam. The explanations are intentionally short, matching what you should be able to produce under time pressure.
1. Reward Transformations

Correct: A and C. Positive scaling preserves ordering. Potential-based shaping preserves optimal policies. Per-step constants in variable-length episodic tasks, nonlinear transforms, and changing γ can change policy ordering.

2. Behavior Cloning And DAgger

False, True, True, False, True. Validation is on expert states, not deployment states. DAgger is supervised learning with online data collection. It needs interaction and expert labeling/intervention.

3. Baselines

Correct: A, B, and D. A baseline is unbiased if it does not depend on the sampled action after conditioning on the state/context. C and E depend on the sampled action or its consequence and can bias the estimator.

4. PPO

Correct: A, B, and D. PPO is not fully off-policy like replay-buffer Q-learning. Clipping limits the surrogate incentive, not the literal parameter change by exactly ε.

5. Discounted Returns

G0 = 0 + 0.5·2 + 0.52(-1) + 0.53·4 = 0 + 1 - 0.25 + 0.5 = 1.25.

G1 = 2 + 0.5(-1) + 0.52·4 = 2 - 0.5 + 1 = 2.5.

6. REINFORCE Update Sign

If action A: g = d/dp log p = 1/p = 2.5. Update: p_new = 0.40 + 0.01·5·2.5 = 0.525.

If action B: g = d/dp log(1-p) = -1/(1-p) = -1/0.6 = -1.667. Update: p_new = 0.40 + 0.01·5·(-1.667) ≈ 0.317. Good B rollout lowers probability of A, making B more likely.

7. Actor-Critic Advantage

Â = 3 + 0.9·8 - 7 = 3.2. Positive advantage, so increase probability of the sampled action.

8. Q-Learning Targets

Best next Q is 7. Target: y = 1 + 0.9·7 = 7.3.

Loss: (9 - 7.3)2 = 2.89.

9. n-Step Target

y = 1 + 0.5·0 + 0.52·2 + 0.53·6 = 1 + 0 + 0.5 + 0.75 = 2.25.

Large n can be biased off-policy because intermediate rewards/actions in the replay trajectory came from the behavior policy, not necessarily the current/target policy.

10. Flow Matching Concept

Squared-error regression can average left and right into straight, which is bad. Flow matching starts from random noise and transforms it into an action, so different noise samples can produce different valid modes such as left or right.

11. DQN Stabilization

Replay buffer decorrelates samples and reuses experience. Target network makes bootstrap labels change slowly. Exploration collects coverage of more state-action pairs so Q-learning is not trained only on early bad behavior.
